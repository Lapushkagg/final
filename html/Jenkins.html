Jenkins

  • Jenkins — это инструмент для автоматизации сборки, тестирования и развертывания приложений.

Как скачать Jenkins

  • Jenkins создан на базе Java. Это значит, что для работы Jenkins на нашем устройстве должен быть Java. Перейдем в командную строку и введем команду java --version . Она выведет версию Java на компьютере. Если командная строка вывела ошибку, значит, Java на нашем компьютере нет и его нужно установить: https://download.oracle.com/java/17/archive/jdk-17_windows-x64_bin.exe (sha256).
  • Перейдем на страницу скачивания Jenkins: https://www.jenkins.io/download/#downloading-jenkins. В таблице Download Jenkins 2.426.1 LTS выберем свою операционную систему — начнется скачивание.
  • Идем в папку «Загрузки» и запускаем установочный файл.
  • Выбираем параметры установки. Указываем папку на компьютере, в которую сохраним Jenkins.
  • Jenkins установлен. Переходим по адресу https://localhost:8080/.
